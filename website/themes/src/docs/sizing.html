Sizing

Landkit only

Sizing utlities to control the size of an element.

  • .mw-25(50, 75, 100, 110, 120, 130, 140, 150) - controls the max-width of an element in %.
  • .vw-25(50, 75, 100, 110, 120, 130, 140, 150) - controls the width of an element in vw.
<div class="mw-50"></div>
<div class="vw-50"></div>

Responsive

Responsive sizing utlities to control the size of an element at every breakpoint.

<div class="h-sm(md, lg, xl)-25(50, 75, 100, 110, 120, 130, 140, 150)"></div>
<div class="w-sm(md, lg, xl)-25(50, 75, 100, 110, 120, 130, 140, 150)"></div>
<div class="mw-sm(md, lg, xl)-25(50, 75, 100, 110, 120, 130, 140, 150)"></div>
<div class="vw-sm(md, lg, xl)-25(50, 75, 100, 110, 120, 130, 140, 150)"></div>
